This repository contains custom scripts used for the analyses and results visualization presented in the manuscript "The tiniest genomes shrink much further: extreme reductive evolution in planthopper symbionts."

Specifically, it includes:

    A script for the annotation of fast-evolving ancient planthopper bacterial symbionts Sulcia and Vidania

    A script for the visualization of symbiont genome content

    A script for visualizing the contribution of symbionts to amino acid biosynthesis pathways
